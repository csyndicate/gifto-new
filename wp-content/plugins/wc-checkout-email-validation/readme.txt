== Description ==

A lightweight plugin that checks for the most common email typos in the customer's email and provides them with a warning. Using this plugin your customers will be much more likely to enter a valid email address, which will simplify and improve your order management.

**USAGE**
You don't need to configure the plugin, just upload and activate it and it's ready to go.

**CURRENTLY CHECKING FOR THE FOLLOWING EMAIL DOMAINS**

* Gmail
* Hotmail
* Outlook
* Protonmail
* Yahoo

== Frequently Asked Questions ==

= What email domains does it check for? =

So far we have added support for Gmail, Hotmail, Outlook, Yahoo and Protonmail. If you have any common typos for other email domains please send them to [support@b4ecommerce.com](mailto:support@b4ecommerce.com) and we will add them as fast as possible.
